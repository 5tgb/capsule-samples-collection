## Description

This capsule demonstrates how to use Bixby's Video Player

## Usage - Video Player

To see Bixby's Video Player in Action, run

`play video` - plays a video from Vimeo

`play loop` - loops a YouTube video


For more information see - https://bixbydevelopers.com/dev/docs/reference/type/video
